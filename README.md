Impossible_Pack_For_Minecraft
By: Daplow

Datapack Changes:
- Enchanted Diamond armor to zombie, drowned, husk, skeleton, bogged, stray, wither_skeleton
- Makes spiders/cave_spiders invisible
- Creepers are charged
- Gives all hostile mobs sharpness swords/bow/axe
- Enchanted Gold Armor to piglin, piglin_brute, zombified_piglin
- Pillagers and Piglins now use Fireworks in Crossbows

How to play:
1. Download the "Impossible Pack" folder
2. Create a new world in Minecraft
3. Click on "Data Packs" in the "More" section
4. Click on "Open Pack Folder" and drop the "Impossible Pack" folder in
